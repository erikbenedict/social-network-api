# Social Network API

## Description

This is an API for a social network web application where users can share their thoughts, react to friendsâ€™ thoughts, and create a friend list. It's uses a NoSQL database so that a website can handle large amounts of unstructured data.

## Built With

- Express.js
- MongoDB
- Mongoose

## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

- Clone the repo
  ```sh
  git clone git@github.com:erikbenedict/social-network-api
  ```
- Install NPM packages
  ```sh
  npm i
  ```
